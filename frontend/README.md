# 💿 Vinyl: The Social Music Workstation

![Vinyl Social Workstation Mockup](file:///C:/Users/abhij/.gemini/antigravity/artifacts/vinyl_workstation_mockup.png)

> **"Analogue Soul. Neural Intelligence."**

Vinyl is a masterpiece-grade social music workstation designed to bridge the gap between tactile listening and cutting-edge artificial intelligence. It translates emotional queries into curated soundscapes using the power of NVIDIA's Llama 3.1 405B-Instruct model.

---

## ✨ Features that WOW

### 🛡️ **Neural Discovery Engine**
*   **Synaptic Mood Search**: Type how you feel (*"I'm feeling melancholic in a rainy city"*), and our AI parses the emotional resonance to find the perfect track from your library.
*   **NVIDIA Llama 3.1 Integration**: Powered by NVIDIA NIM for ultra-low latency, world-class LLM recommendations.

### 💎 **The Vinyl Vault (3D Gallery)**
*   **Tactile Explorations**: A high-fidelity 3D album gallery with perspective-shift hover effects.
*   **Digital Vinyl Interaction**: Watch as vinyl discs peek out from their sleeves as you explore your masterpiece collection.
*   **The Collection Creator**: Personalize your studio with custom-named playlists and curated stacks.

### 🎭 **The Analogue Player**
*   **Spinning Reality**: A high-fidelity vinyl turntable interface where the tonearm tracks the needle's progress across real grooves.
*   **Physics-Based Interaction**: Draggable 6px progress bars with "Magnetic" seeking precision.

### 🤝 **Social Synchronicity**
*   **Synced Listen-Alongs**: Real-time websocket integration allows you to sync playback with friends across the globe.
*   **Live Blend Rooms**: Experience music together in low-latency holographic rooms.

## 🚀 Tech Stack

- **Frontend**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) + [Anime.js](https://animejs.com/)
- **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) + [Glassmorphism](https://glassmorphism.com/)
- **Intelligence**: [NVIDIA NIM](https://www.nvidia.com/en-us/ai-data-science/generative-ai/nim/) (Llama 3.1 405B)
- **Real-time**: [Socket.io](https://socket.io/)
- **Backend**: [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)

---

## 🛠️ Performance-First Architecture

We prioritize "Tactility" and "Immediacy." The UI is optimized for zero-layout-shift and uses hardware-accelerated animations to ensure that even with heavy 3D effects, the experience remains buttery smooth at 60fps.

## 🎧 Getting Started

1.  **Clone the workstation**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/vinyl-social-workstation.git
    ```
2.  **Initialize the neural core**:
    ```bash
    cd vinyl-social-workstation/backend
    npm install
    # Set your NVIDIA_API_KEY in .env
    npm run dev
    ```
3.  **Fire up the visual deck**:
    ```bash
    cd ../Music Player App
    npm install
    npm run dev
    ```
